## 缺陷汇总

### 🔴 P0 (功能阻断/数据风险)
1. progressRate无0-100范围校验
2. 项目名称无最大长度限制
3. 用户创建无应用层输入校验

### 🟡 P1 (体验/规范)
4. 重复邀请提示"数据约束冲突"→应提示"成员已存在"
5. 编辑不存在的项目→500非400
6. 不存在填报审核→500应返回404
7. engineer创建支持→403(业务规则待定)
8. HTTP Status与JSON code不一致(部分仍200)

### 🔵 P2 (需浏览器/待验证)
9. UI-001~019全部需浏览器
10. EDGE-025~026引用完整性需SQL验证
