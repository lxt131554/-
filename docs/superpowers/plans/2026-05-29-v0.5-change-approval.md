### Task 1: SQL + Entity + Mapper + Service + Controller

**sql/v0.5.sql:**
```sql
USE pm_system;
CREATE TABLE IF NOT EXISTS sys_change (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    project_id BIGINT NOT NULL,
    content TEXT COMMENT '变更核心内容',
    confirm_time DATE COMMENT '与业主确认变更时间',
    impact TEXT COMMENT '变更影响范围',
    status VARCHAR(20) DEFAULT 'pending',
    create_user_id BIGINT,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP
) COMMENT '变更记录表';

CREATE TABLE IF NOT EXISTS sys_approval (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    project_id BIGINT NOT NULL,
    review_situation TEXT COMMENT '评审情况',
    fail_reason TEXT COMMENT '未通过原因',
    confirm_time DATE COMMENT '最终确认时间',
    create_user_id BIGINT,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP
) COMMENT '成果评审与审批表';
```

**Entities:** SysChange, SysApproval — 标准 @TableName + @TableId(AUTO) + @TableField(exist=false) projectName

**Mappers:** extends BaseMapper

**Services:**
- SysChangeService: listByProject, create, confirm (set status='confirmed'+confirmTime)
- SysApprovalService: getByProject, saveOrUpdate

**Controllers:**
- ChangeController: GET/POST /api/projects/{projectId}/changes, PUT /api/changes/{id}/confirm
- ApprovalController: GET/POST /api/projects/{projectId}/approval
