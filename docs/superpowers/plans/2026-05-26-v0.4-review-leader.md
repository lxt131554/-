### Task 1: SQL + 实体 + Mapper

**sql/v0.4.sql:**
```sql
USE pm_system;

CREATE TABLE IF NOT EXISTS sys_review (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    project_id BIGINT NOT NULL COMMENT '项目ID',
    overall_deviation TEXT COMMENT '项目整体执行偏差',
    efficiency_rating TEXT COMMENT '项目综合效率评价',
    quality_rating TEXT COMMENT '项目质量评价',
    communication_note TEXT COMMENT '全过程沟通情况',
    create_user_id BIGINT COMMENT '填报人ID',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP
) COMMENT '项目自评表';

CREATE TABLE IF NOT EXISTS sys_experience (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    project_id BIGINT NOT NULL COMMENT '项目ID',
    reusable_experience TEXT COMMENT '值得复用的经验',
    shortcomings TEXT COMMENT '相关短板或缺陷',
    risks TEXT COMMENT '风险点',
    improvement TEXT COMMENT '改进建议',
    create_user_id BIGINT COMMENT '填报人ID',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP
) COMMENT '经验总结表';
```

**Entity + Mapper:** 标准 MyBatis-Plus 模式，2个实体+2个Mapper。实体加 @TableField(exist=false) 的 projectName 透传字段。

### Task 2: Service + Controller

**SysReviewService:** extends IService<SysReview>
- `SysReview getByProjectId(Long projectId)` — 查某个项目的自评
- `SysReview saveOrUpdateReview(SysReview review, Long userId)` — 保存或更新（一个项目只有一份自评）

**SysExperienceService:** extends IService<SysExperience>
- `SysExperience getByProjectId(Long projectId)`
- `SysExperience saveOrUpdateExperience(SysExperience exp, Long userId)`
- `List<SysExperience> listAll()` — 经验库用

**ReviewController:** `/api/projects/{projectId}/review`
- GET → 获取自评
- POST → 保存/更新自评

**ExperienceController:** 
- GET `/api/projects/{projectId}/experience` → 获取项目经验总结
- POST `/api/projects/{projectId}/experience` → 保存/更新
- GET `/api/experiences` → 经验库列表（所有已完成项目）

**LeaderDashboardController:** GET `/api/leader-dashboard`
- 返回：{ activeProjects: N, completedProjects: N, openDeviations: N, pendingSupports: N, projects: [...] }
- projects 数组含每个项目的名称、状态、最新阶段、偏差数、支持数
